# ROADMAP — Recruiting Intelligence Platform

> Текущее состояние: client-only SPA (React 19 + Vite 8 + Dexie/IndexedDB + Zustand)
> 18 страниц, 10 таблиц, 8 сторов, 0 тестов, 0 CI/CD
> Дата: Апрель 2026

---

## Фаза 0 — Стабилизация MVP (2–3 недели)
**Цель:** Довести текущий прототип до production-ready состояния

### Задачи
- [ ] Покрытие unit-тестами ядра (matching, aggregation, roadmap) — Vitest
- [ ] E2E-тесты критических сценариев (регистрация, создание вакансии, матчинг) — Playwright
- [ ] CI/CD pipeline (GitHub Actions: lint → typecheck → test → build → deploy)
- [ ] Деплой статики (Vercel / Cloudflare Pages)
- [ ] Аудит доступности (a11y) и мобильной адаптации
- [ ] Документация API сущностей и архитектуры
- [ ] Исправление накопленных TS-ошибок (TreePicker, VacancyDetail, responseStore)

### Команда
| Роль | Кол-во | Зона ответственности |
|------|--------|---------------------|
| **Fullstack-разработчик (Middle+)** | 1 | Тесты, CI/CD, фикс TS-ошибок, деплой |
| **Основатель / продакт** | 1 | Приоритизация, ревью, документация |

**Итого: 2 человека**

## Фаза 1 — Серверная часть и мультитенантность (2–3 месяца)
**Цель:** Перейти от локального хранилища к облачному, многопользовательскому сервису

### Задачи
- [ ] Backend API (Node.js / Go / Rust) — REST или tRPC
- [ ] PostgreSQL — миграция схемы данных с Dexie
- [ ] JWT-аутентификация, OAuth (Google, GitHub)
- [ ] Миграция клиента: Dexie → API-слой (React Query / SWR)

- [ ] Мультитенантность (организации/команды)
- [ ] Хранилище файлов (резюме, аватары) — S3-совместимое

- [ ] Мониторинг (Sentry, Prometheus + Grafana)

### Команда
| Роль | Кол-во | Зона ответственности |
|------|--------|---------------------|
| **Backend-разработчик (Senior)** | 1 | API, БД, авторизация, мультитенантность |
| **Fullstack-разработчик (Middle+)** | 1 | Миграция клиента на API, React Query |
| **DevOps-инженер (Middle)** | 1 | Инфраструктура, CI/CD, мониторинг, S3 |
